BALOK

Rumus:

Luas = 2 x [(p x l) + (p x t) + (l x t)]

Keliling = 4 x (p + l + t)

Volume = p x l x t

KUBUS

Rumus:

Luas = 6 x s

Keliling = 12 x s

Volume = s x s x s

PRISMA SEGITIGA

Rumus:

Luas = (2 x luas alas) + (keliling alas x tinggi prisma)

Keliling = (2 x keliling alas) + (3 x keliling sisi)

Volume = [(alas x tinggi) : 2] x tinggi prisma

BOLA

Rumus:

Volume = 4/3 x π x r^3

Luas = 4 × π × r²

Keliling alas = 2 x π x r atau π x d

LIMAS SEGITIGA

Rumus:

Volume = ⅓ × La × t atau V = ⅓ × (½ × as × ts) × t

LIMAS SEGI EMPAT

Rumus:

Luas = 4 x (1/2 x a x t) + (s x s)

Volume = 1/3 x luas alas x tinggi

Keliling = 4 x s

TABUNG

Rumus:

Luas = luas lingkaran = πr^2

Keliling = 2 x π x r atau π x d

Volume = luas alas (lingkaran) x tinggi = (π x r2) x t
